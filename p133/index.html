R(km) (k > 1) は R(m) の倍数である。

証明

R(k) は以下のように定義される。

R(k) = (10^k - 1) / 9

上記より、

R(km) / R(k) = (10^(km) - 1) / 9R(a)

である。ここで

9R(k) + 1 = 10^k

と書けることから、両辺を m 乗して

10^(km) = (9R(k) + 1)^m

を得る。上記を代入することで以下を得る。

R(km) / R(k) = ((9R(k) + 1)^m - 1) / 9R(a)

右辺は整数であるため、R(km) は R(k) で割り切れる■
